Selected Open Source Projects

  • TideGAN: Audio timbre transfer using CycleGAN with WaveGAN-style generators and discriminators.

  • PyTorch WaveGAN implementation: An implementation of WaveGAN using PyTorch.

  • Identifying and reducing gender bias in word-level language models: Code for analyzing gender bias in text corpora and training word-level RNN language models with a regularizer that penalizes gender bias in the embedding layer.

  • Deep beat tracking and tempo estimation: Beat tracking and tempo estimation using deep learning, using raw audio as the input to the network. Based off of methods from Sebastian Böck et al.

  • audiosetdl: Scripts for downloading the AudioSet dataset from Google.

  • L3 Embedding: Implementation of the Look, Listen, Learn model and models for the down-stream task of urban sound classification. Part of the SONYC project.

  • Stream instrument source separation: PLCA-based source separation of musical instruments on streamed audio.

  • MarkovMixer: Real-time Markov chain mashup program, with transition probabilites based on cross-correlation similarity. Made as a starter template for EE 126 class projects.

  • fortissimo: Simple programming language for music composition, implemented in Python and with Overtone as the synthesis backend.

  • PandaPhone: Ambient generative instrument using a Leap Motion controller and Max MSP. Hand gestures modify note and velocity transition probabilities learned from MIDI data.
  • Augmented reality digital audio workstation: Hackathon proof-of-concept for synthesizers controlled by augmented reality knobs.
